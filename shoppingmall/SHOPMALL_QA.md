# SHOPMALL QA 체크리스트

20252361 김지연
기능 설명: 상품 검색, 찜, 장바구니, 주문 완료 처리 수동 QA 문서

| 번호 | 기능 | 테스트 절차 | 예상 결과 | 결과 |
|---:|---|---|---|---|
| 1 | 상품명 검색 | 메인에서 `셔츠` 입력 | 상품명에 셔츠가 포함된 카드만 표시 | 미확인 |
| 2 | 브랜드명 검색 | 메인에서 `BASIC` 입력 | BASIC HOUSE 상품이 표시 | 미확인 |
| 3 | 없는 검색어 | 메인에서 `없는상품` 입력 | 검색 결과 없음 문구 표시 | 미확인 |
| 4 | 카테고리 선택 | 신발 카테고리 클릭 | 신발 상품만 표시 | 미확인 |
| 5 | 가격 낮은 순 | 정렬에서 가격 낮은순 선택 | 낮은 가격부터 정렬 | 미확인 |
| 6 | 가격 높은 순 | 정렬에서 가격 높은순 선택 | 높은 가격부터 정렬 | 미확인 |
| 7 | 평점 높은 순 | 정렬에서 평점 높은순 선택 | 평점 높은 상품 우선 표시 | 미확인 |
| 8 | 검색과 정렬 동시 적용 | 검색어 입력 후 가격 정렬 선택 | 검색 결과 안에서 정렬 적용 | 미확인 |
| 9 | 찜 추가 | 상품 하트 클릭 | 하트가 채워지고 찜 배지 증가 | 미확인 |
| 10 | 찜 해제 | 같은 하트 다시 클릭 | 하트가 비고 찜 배지 감소 | 미확인 |
| 11 | 새로고침 후 찜 유지 | 찜 추가 후 새로고침 | 찜 상태와 배지가 유지 | 미확인 |
| 12 | NEW 상품 찜 목록 표시 | NEW 상품 찜 후 찜 목록 이동 | NEW 상품이 찜 목록에 표시 | 미확인 |
| 13 | 찜 목록에서 장바구니 추가 | 찜 목록 상품의 장바구니 담기 클릭 | 장바구니 배지 증가 | 미확인 |
| 14 | 동일 상품 중복 추가 | 같은 상품 장바구니 담기 2회 클릭 | 새 행이 아니라 수량 2로 처리 | 미확인 |
| 15 | 수량 증가·감소 | 장바구니에서 +/- 클릭 | 수량과 금액이 즉시 재계산 | 미확인 |
| 16 | 상품 삭제 | 장바구니에서 삭제 클릭 | 해당 상품 행 제거 | 미확인 |
| 17 | 장바구니 전체 삭제 | 전체 삭제 클릭 | 장바구니 비움 안내 표시 | 미확인 |
| 18 | 49,999원 배송비 | 총 상품금액 49,999원 상태 확인 | 배송비 3,000원 | 미확인 |
| 19 | 50,000원 배송비 | 총 상품금액 50,000원 상태 확인 | 배송비 0원 | 미확인 |
| 20 | 50,001원 배송비 | 총 상품금액 50,001원 상태 확인 | 배송비 0원 | 미확인 |
| 21 | 주문 필수값 누락 | 주문서 필수값 비우고 제출 | 첫 오류 메시지 표시 및 focus 이동 | 미확인 |
| 22 | 잘못된 전화번호 | `1234` 입력 후 제출 | 전화번호 형식 오류 표시 | 미확인 |
| 23 | 잘못된 이메일 | `abc` 입력 후 제출 | 이메일 형식 오류 표시 | 미확인 |
| 24 | 정상 주문 | 모든 필수값과 결제 방법/동의 입력 후 제출 | 서버 주문 처리 후 완료 페이지 이동 | 미확인 |
| 25 | 주문 완료 페이지 | 정상 주문 후 완료 화면 확인 | 주문번호, 시간, 상품, 금액, 주소 출력 | 미확인 |
| 26 | 장바구니 초기화 | 주문 완료 후 장바구니 이동 | 장바구니가 비어 있음 | 미확인 |
| 27 | 찜 목록 유지 | 주문 완료 후 찜 목록 이동 | 찜 목록은 유지 | 미확인 |
| 28 | 잘못된 LocalStorage JSON | 개발자도구에서 cart/wishlist 값을 잘못된 JSON으로 변경 후 새로고침 | 화면 중단 없이 빈 상태로 복구 | 미확인 |
| 29 | 브라우저 콘솔 오류 | 메인, 찜, 장바구니, 주문서, 완료 페이지 순회 | JavaScript 오류 없음 | 미확인 |
| 30 | Tomcat 컨텍스트 경로 배포 | `/shoppingmall` 컨텍스트로 WAR 배포 | CSS/JS/링크/form 경로 정상 동작 | 미확인 |
